# Go Learn Project

> Simple REST API using Go, Chi, and Redis.

## Stack

- **Golang**
- **Chi** (HTTP router)
- **Redis** (as database)

## Getting Started

1. Clone this repo
2. Install dependencies
   ```sh
   go mod tidy
   ```
3. Run Redis (e.g. via Docker)
4. Run the app:
   ```sh
   go run main.go
   ```
